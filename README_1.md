
## C、C++是高性能语言, golang/java是内存安全语言

## drogon优势
+ 比较健全的中文帮助文档
+ 最快的web应用服务器框架

## Drogon vs nginx
+ Drogon是高性能web框架，可以基于框架鞋任何应用
+ nginx是Web服务器, 单独nginx只是流量转发，但可以使用lua在openresty上写应用(nginx + lua)
